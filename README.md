# SakaChloFont  

 
hololive所属のVtuver 沙花叉クロヱ さんの手書き風フォント  

全角ひらがな、カタカナのみです  
英数字、漢字は一部だけ入っています  
カタカナの一部はその他の文字の組み合わせて編集されたものがあります  

収録されている文字は chrlist.txt を参照  



配信画面のトレース、及びその編集で作成された物なので  
COVER Corp. 二次創作ガイドライン に従ってください  
https://www.hololive.tv/terms  

フォントファイルの改変や再配布は不可
